## Project Overview

Led the design and development of a skier safety system to increase visibility on terrain park jumps. Designed and 3D printed all CAD components using Fusion 360, including mounts for LEDs, batteries, and sensors. Developed the hardware layout and electronics integration plan using Raspberry Pi, and contributed to programming logic. Presented the concept at the end of the class to a local VC company.
